This package duplicates some of the HBase APIs to make it easier to work with the transaction APIs. They are merely a utility, i.e., they are not required for using the transaction manager. The main reasons they exist are the following:
  1. Explicitly remove the timestamp from the APIs --> HTX uses timestamps to implement MVCC
  2. Convert Deletes into Put + delete flag --> HTX handles deletion of versions when they are no longer needed
  3. Restrict the APIs to the ones we have tested
